# Pet Finder

Proyecto del módulo 7 para la carrera de apx.school

## En este proyecto se uso

### Cliente

- Vite
- Arquitectura orientada a componentes
- Custom HTML Components
- Typescript
- Axios
- TailwindCss
- Mapxbox para buscar y mostrar puntos en el mapa

### Servidor

- Arquitectura MVC
- Expressjs
- PostgreSql
- Typescript
- Algolia para almacenar la geolocalización
- Sendgrid para el envio de emails
- Cloudinary para host de imagenes
